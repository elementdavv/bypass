## Domain match rule. 域名匹配规则

Match the current domain name and all its subdomains. 匹配當前域名及其所有子域名:

-   `cn`: `cn`, `xxx.cn`, `xxx.xxx.cn`
-   `a.com`: `a.com`, `xxx.a.com`, `xxx.xxx.a.com`
